# nbsp-adder
Skript přidá do textu na potřebná místa html entitu pro nedělitelnou mezeru: `&nbsp;`. Zatím (28.&nbsp;12.&nbsp;2025) skript přidává HTML entitu pouze před jednompísmnenná slova.

Další případy, kdy se do textu přidává nedělitelná mezera: https://cs.wikipedia.org/wiki/Nezlomitelná_mezera#Využití.
